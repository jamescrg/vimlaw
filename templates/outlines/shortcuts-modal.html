Keyboard Shortcuts

Navigation & Selection
↑ / ↓ Move focus up/down
Shift + ↑ / ↓ Extend selection
Shift + Click Select range
Enter Edit focused item
Editing
Enter Create new item below
Backspace Delete empty item
Escape Cancel editing
Indentation
Tab Indent item
Shift + Tab Outdent item
Reordering
Ctrl + ↑ Move item up
Ctrl + ↓ Move item down
Ctrl + Delete Delete item
Formatting
Ctrl + Shift + H Toggle heading
Ctrl + ; Add source
Alt + Y/G/R/P/O Highlight text in color
Alt + C Clear highlight from text
